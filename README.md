# 🛒 Supermarket Management System — Java OOP

A console-based Supermarket Management System developed using Java and Object-Oriented Programming principles.

## 🚀 Features
- Product management (Add / Update / Delete)
- Inventory tracking
- Billing and checkout system
- Structured class design using OOP
- User-friendly console interaction

## 🧠 OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## 🛠️ Technologies
- Java
- OOP
- Data Structures
- IntelliJ IDEA / Eclipse

## ▶️ How to Run
1. Clone the repository  
2. Open in your Java IDE  
3. Run the Main class
